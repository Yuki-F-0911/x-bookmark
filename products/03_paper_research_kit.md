# 📚 海外論文リサーチ自動化キット

> 指定テーマの最新論文を自動検索→一流教授レベルの構造化レビューを自動生成。
> R&D部門・研究者・専門家の情報収集を10倍速にします。

## この商品で得られるもの

### 1. 論文検索プロンプト（Google Scholar / arXiv対応）

```
## 検索戦略プロンプト

あなたは学術リサーチの専門家です。以下のテーマについて、
効果的な検索クエリを生成してください。

テーマ: {研究テーマ}
対象期間: {例：過去2年}
対象分野: {例：運動生理学、AI、バイオテック}

## 出力
1. 英語検索クエリ（3〜5パターン）
2. 各クエリの意図（何を見つけたいか）
3. 除外すべきキーワード
4. 推奨ジャーナル/カンファレンス
```

### 2. 論文レビュープロンプト（6段階査読フロー）

| ステップ | 内容 |
|---------|------|
| 1. 概要把握 | タイトル・アブストラクトから研究の位置付けを判定 |
| 2. 方法論評価 | 研究デザイン・統計手法の妥当性を評価 |
| 3. 結果分析 | 主要な発見と効果量の解釈 |
| 4. 限界点特定 | 著者が述べていない限界も含めて指摘 |
| 5. 実践的示唆 | 「この研究結果をどう使うか」の提案 |
| 6. 総合評価 | A〜Dランクでの格付けと推薦度 |

### 3. 構造化レビューレポートテンプレート

```markdown
# 論文レビュー: {論文タイトル}

## 基本情報
- 著者: {著者名}
- ジャーナル: {掲載誌}
- 発表年: {年}
- DOI: {DOI}

## エグゼクティブサマリー（3文以内）
{この論文が何を発見し、なぜ重要か}

## 研究の強み
1. {強み1}
2. {強み2}

## 研究の限界
1. {限界1}
2. {限界2}

## 実践への示唆
{この知見をビジネス/実務にどう活かせるか}

## 総合評価: {A/B/C/D}
```

### 4. Excel一覧表生成プロンプト

検索結果を以下のカラムで整理：
- タイトル（英語/日本語）
- 著者
- 年
- 引用数
- アブストラクト要約
- 関連度スコア
- DOIリンク

### 5. Zotero連携ガイド

論文管理ツールZoteroとの連携方法：
- APIを使った自動追加
- コレクション自動整理
- タグ付け自動化
