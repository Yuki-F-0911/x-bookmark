# 🤖 AI情報収集・自動キュレーション完全パッケージ

> XブックマークからAIが自動でニュースレター原稿を生成する仕組みを、あなたのPCに構築できます。

## この商品で得られるもの

### 1. 自動情報収集プロンプト
毎日のX（旧Twitter）ブックマークから、指定テーマに関連する記事を自動分類・スコアリングするClaude APIプロンプト。

```
使用モデル: Claude Sonnet 4.5（コスト: 約$0.02/回）
処理件数: 30件/回（カスタマイズ可）
出力: JSON形式（カテゴリ・重要度・要約）
```

### 2. テーマ別分類プロンプト（カスタマイズ可）

以下の分類軸をあなたのテーマに合わせて変更できます：

```python
# カテゴリ定義（これを書き換えるだけ）
CATEGORIES = [
    "生成AI・LLM動向",
    "SaaS・プロダクト戦略",
    "UX・UIデザイン",
    "開発・エンジニアリング",
    "ビジネス・資金調達",
]
```

### 3. ニュースレター記事生成プロンプト

収集した情報を「読者が課金したくなる」記事に変換するプロンプト。
以下の構成を自動生成します：

- **🔥 今日のトップ3**: 最重要ニュースの深掘り
- **📊 トレンドウォッチ**: マクロトレンド分析
- **💡 編集部の視点**: 独自考察
- **📎 リンク集**: カテゴリ別整理

### 4. Web検索エンリッチメントプロンプト

ツイート本文だけでは不十分な情報を、自動でWeb検索して補足するプロンプト。
「なぜこれが重要か」の文脈を追加します。

### 5. 実装ガイド（Python）

- 環境構築手順（Windows / Mac対応）
- タスクスケジューラーでの自動実行設定
- Slack通知の設定方法
- Note / Substack への投稿フォーマット

## プロンプト詳細

### 要約・分類プロンプト

```
以下のXブックマーク一覧を分析し、**{あなたのテーマ}**というテーマの
特化型ニュースレターの原稿素材として抽出してください。

各ブックマークについて以下の3点を判定してJSON配列で返してください:
1. importance: "high"（重要ニュース・実用的ノウハウ）/ "normal"（一般的な情報）/ "low"（対象外）
2. category: 上記カテゴリのいずれかに分類
3. summary: 読者にとって「なぜ重要か（示唆・インサイト）」を含めた2〜3文の要約

JSONのみで返してください:
[{"id":"ID","importance":"high/normal/low","category":"カテゴリ","summary":"要約"}]
```

### 記事生成プロンプト

```
あなたはテック系有料ニュースレターの編集長です。
以下のブックマーク分析データから、{ターゲット読者}向けの
高品質なニュースレター記事を生成してください。

## 記事要件
- 読者: {ターゲット読者の詳細}
- トーン: 専門的だが親しみやすい
- 構成: トップ3 → トレンド分析 → 編集部考察 → リンク集
- 文字数: 2000〜3000字

## ブックマークデータ
{JSON データ}
```

### キーワード抽出プロンプト（エンリッチメント用）

```
以下のツイートから、Web検索で補足情報を得るための
検索キーワードを2〜3個抽出してください。

ツイート: {ツイート本文}

条件:
- 固有名詞（サービス名、人名、企業名）を優先
- 一般的すぎるキーワードは避ける
- JSON配列で返す: ["keyword1", "keyword2"]
```
