## サービス概要

御社の社内文書・マニュアル・議事録・チャット履歴を全学習させた
**「高精度な社内専用AIアシスタント」** を構築・納品します。

### 従来のチャットボットとの違い

| 従来のFAQボット | 当社の社内ブレイン |
|----------------|------------------|
| 事前にQ&Aを登録 | 文書を入れるだけで学習 |
| 登録外の質問に回答不可 | 文書の内容から推論して回答 |
| メンテナンスに工数大 | 文書追加で自動的に学習 |
| 的外れな回答が多い | 出典を明示した正確な回答 |

### 技術的な強み

1. **ハルシネーション抑制**: 文書に基づいた回答のみ生成。根拠のない推測は「情報が見つかりませんでした」と正直に回答
2. **出典明示**: 回答の根拠となった文書名・ページを自動表示
3. **マルチソース対応**: Notion / Google Drive / SharePoint / PDF / チャット履歴を横断検索

## 技術構成

```
┌─────────────┐     ┌──────────────┐     ┌───────────────┐
│ 社内文書    │ ──→ │ NotebookLM   │ ──→ │ Slack Bot     │
│ (Notion/    │     │ (RAG Engine) │     │ (ユーザーIF)  │
│  Drive/PDF) │     │              │     │               │
└─────────────┘     └──────────────┘     └───────────────┘
                           ↑
                    ┌──────┴──────┐
                    │ Claude API  │
                    │ (回答生成)  │
                    └─────────────┘
```

### セキュリティ

- データは御社のクラウド環境内で処理（外部送信なし）
- アクセス権限はSlackのチャンネル権限に準拠
- 監査ログ: 全質問・回答を記録
- NDA締結済みで対応
