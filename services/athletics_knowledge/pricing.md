# Athletics Knowledge Service - マネタイズ設定

陸上競技に関する最新研究や専門的知識を海外層へ提供するサービスの料金プランおよび収益化戦略。

## 料金プラン (Pricing Plans)

### 1. Basic Plan (Free)

- 最新の陸上関連ニュースとヘッドラインの閲覧
- 各トピックの簡単なAIサマリー（月5回まで生成可能）
- 広告表示あり

### 2. Pro Plan ($15 / month)

- マラソンやスプリントなど、生成されたプレミアムダイジェスト記事の無制限アクセス
- Web検索＋Xツイート＋Google Scholar論文の統合分析
- 広告なし、クリーンなUI
- ブックマーク・検索履歴に基づくパーソナライズ推薦

### 3. Coach / Enterprise Plan ($50 / month)

- Pro Planの全機能
- 特定の選手、地域、ニッチなトレーニング理論に基づくカスタムレポートの自動生成タスク追加
- 記事のPDF/CSVエクスポート機能
- Priority API アクセス（最速での情報収集）

## 戦略ポイント

- 競合が少ない海外の「陸上競技コーチ」「コアなデータファン」層にターゲットを絞り、高いLTV（顧客生涯価値）を維持する。
- プレミアム感のあるUI (Vite/Vanilla, Dark Theme) により、単なる情報の羅列ではなく「分析ツール」としての価値を持たせる。
